# Coverage(Notas)
Não obtivemos coverage de 100%, uma vez que é influenciado por diversos fatores:
-Presença de dead codes nas classes que herdam características da classe ship;
-Verificações desnecessárias em alguns métodos;
-A classe Tasks, faz com que a média geral de coverage diminua, ficando a 70%. Contudo a verdadeira percentagem do coverage sem o Tasks é de 89,94%, ou seja 90%, o que demosntra que o nosso código está bem testado.
